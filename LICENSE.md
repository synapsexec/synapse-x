❌ **Injection Fails on Windows**
The executor cannot attach to the target process, and an error message appears in the console.
**Solution:** Run **Synapse X Executor** as Administrator by right-clicking the `.exe` file. Also verify that your antivirus has not quarantined any core files from `C:\Program Files\SynapseX\`.

❌ **Scripts Not Executing**
The script loads in the editor but nothing happens when you press Execute.
**Solution:** Check the debug console for syntax errors. Ensure `script_timeout` in `config.json` is set high enough for longer scripts. Confirm that the target application is fully loaded before injection.

❌ **Synapse X Executor Ui Not Rendering Correctly**
Panels appear overlapping, fonts are missing, or the theme does not apply.
**Solution:** Delete the `ui_cache` folder from your installation directory and restart. On MacOS, check that your display scaling is set to default in `System Preferences > Displays`.

❌ **Android .apk Crashes on Launch**
The application closes immediately after the splash screen on Android devices.
**Solution:** Ensure your device runs Android 8.0 or higher. Clear existing app data through `Settings > Apps > SynapseX > Clear Data`. Reinstall the `.apk` from the official source.

❌ **Auto-Update Stuck in Loop**
The executor restarts repeatedly trying to apply an update.
**Solution:** Manually download the latest build from the repository. Delete the `update_cache` folder inside your installation directory before launching the new version.

❌ **High Memory Usage During Execution**
The system becomes slow or unresponsive while scripts are running.
**Solution:** Lower the `memory_limit` value in `config.json` and reduce `max_threads` to `2`. Close unnecessary background applications. Consider upgrading your RAM if you consistently run heavy scripts through **Synapse X Executor**.
